# Descripción

Esta base de datos tendrá los siguientes datos de los estudiantes:

- ID
- Nombre
- Apellido
- Cédula
- Ciudad
- Teléfono
- Correo
- Carrera

Será estará en una base de datos MySQL y se podrán realizar las siguientes acciones en la base de datos desde la página:

- Insertar
- Consultar
- Modificar
- Eliminar
